# Background sounds

Drop the following looping audio files here for the Meditation page:

- `ocean.mp3`
- `rain.mp3`
- `birds.mp3`

Any royalty-free ambient loops work (e.g. from Pixabay / Freesound). The
players reference these exact filenames. If a file is missing, the player
surfaces a friendly "sound unavailable" message instead of crashing.
